0.7.1.ALPHA

Update System

In this release, we've added the much-anticipated update feature! You may have noticed that we started making it back in version 0.6, but didn't finish it for some reason (including slow GitHub downloads, etc.). Now, we can check for XMCL updates and download and install them.More see XMCLServer

Bug Fix

中文版:
修复部分整合包无法修改启动时游戏窗口大小的问题
修复终止游戏后启动器不会停止等待的问题
修复 1.5.2 及以下版本不能下载资源文件的问题
修复某些情况下启动游戏时，启动器会崩溃的问题
修复进入游戏设置后无法重命名部分游戏版本的问题
修复不能在 Java 17 下载 Forge 和自动更新的问题
修复重命名版本时，输入不合法的新名字后的错误提示
修复重命名被依赖的版本后，会破坏其他依赖这个版本的版本的问题
修复在自定义游戏运行目录时，输入不合法的路径会导致崩溃的问题
修复在输入特定错误的 authlib-injector 服务器地址时会导致启动器崩溃的问题

English version:
Fix the problem that some packages can't change the size of the game window at startup.
Fix the issue where the launcher doesn't stop waiting after game termination
Fix the problem that resource files cannot be downloaded from version 1.5.2 and below.
Fix the issue where the launcher crashes when starting the game in some cases
Fix the problem of not being able to rename part of the game version after entering game settings.
Fix issue with Forge and automatic updates not downloading in Java 17
Fix error when renaming a version after entering an illegal new name
Fix issue where renaming a dependent version would break other versions that were dependent on that version
Fix the issue where entering an illegal path caused a crash when customizing the game run directory
Fix an issue that causes the launcher to crash when typing in a specific wrong authlib-injector server address
